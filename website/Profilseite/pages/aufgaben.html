2. Uebung Repetition Responsive Website

Ziele Responsive Webseite wildlife.html
Zeit 30 Minuten
Ausgangslage Wildlife.html mit eingebundenem Video.
Aufgabe
  • Ändern Sie den Code demensprechend, dass die Seite responsive wird und Smartphones speziell begrüsst.
  • Erweitern Sie den Headbereich mit dem speziellen Metatag, welches die Skalierung auf die Endgeräte anpasst. <meta viewport……>
  • Ergänzen Sie das CSS für den Video tag, damit das Video nur immer maximal 100% Breite einnimmt.
  • Integrieren Sie media queries für die Screengrösse min-width: 320px and max-width: 480px
  • Der Titel „Welcome Smartphone soll nur für die oben definierten Screens erscheinen. Navigation soll wieder untereinander dargestellt sein und die Farbe wechseln.
